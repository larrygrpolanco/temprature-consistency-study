[1b] This is achieved by nesting an item within itself or within another item of the same kind.  
[1b] Recursive patterns in everyday life are ubiquitous and include, for example, computer folders stored inside other computer folders, Russian dolls nested in each other, Romanesco broccoliâ€™s spirals arranged in a spiral, and the same number of minutes passed within the same number of hours (e.g., 12:12).
